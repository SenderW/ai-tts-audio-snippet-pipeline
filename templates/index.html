AI TTS Audio Snippet Pipeline

This small tool takes an MP3 file, generates four text to speech snippets and inserts them into the audio. It is a simple example for AI based automation in podcast and audio workflows.

How this tool works

The server expects at least four lines of text. The first four non empty lines are used as:

  • Line 1 - intro before the audio
  • Line 2 - first mid roll after the first quarter of the track
  • Line 3 - second mid roll after the second quarter of the track
  • Line 4 - outro after the end of the track

For each of these lines the app calls OpenAI text to speech, creates short speech snippets and joins them with your uploaded MP3 file. Then it writes simple ID3 tags and returns a new MP3 file for download.

The app will use only the first four non empty lines. Additional lines are ignored.

This tool modifies audio files that you upload. Check the box above only if you own the rights or have permission to process the material.
